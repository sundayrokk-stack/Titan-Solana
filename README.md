# Telegram Solana Trading Bot

A professional Telegram trading bot with Solana-style UI/UX, ready for deployment on Render.com.

## Features

- ✅ Professional UI with emojis and formatted text
- ✅ Three-stage onboarding flow
- ✅ Grid-based trading menu
- ✅ Risk warning and introduction
- ✅ Dummy wallet generation
- ✅ Render.com ready with Flask server
- ✅ Clean message editing (no spam)
